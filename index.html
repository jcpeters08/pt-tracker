One-time setup — add your GitHub token

This app reads your routines from the GitHub repo and writes back logged sessions. It needs a personal access token scoped to jcpeters08/pt-tracker.

  1. Open github.com/settings/tokens?type=beta → Fine-grained PAT.
  2. Repository access: Only select repositories → jcpeters08/pt-tracker.
  3. Permissions → Repository → Contents: Read and write.
  4. Paste the PAT below and click Test & save. It is encrypted at rest with a server-side key on the auth Worker.
